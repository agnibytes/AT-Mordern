## 🛠️ TECH STACK & ARCHITECTURE

The project is built on the absolute cutting-edge, favoring performance as much as it favors aesthetic.

### ✦ Core Frameworks
*   **Next.js 16 (App Router)**: The architectural spine. Fast, future-proof, and optimized for SEO.
*   **React 19**: The latest concurrent engine, delivering rock-solid stability for complex UI states.
*   **Tailwind CSS**: Precision styling used to craft refined, high-fashion interfaces.

### ✦ Motion & Physics
*   **GSAP (GreenSock)**: The industry standard for high-performance timelines and scroll-triggered storytelling.
*   **Framer Motion**: Used for declarative micro-interactions and seamless layout transitions.
*   **Lenis**: The smooth-scroll layer that ties the entire tactile experience together.

### ✦ 3D & Icons
*   **Three.js & @react-three/drei**: Subtle spatial elements and shader-based backgrounds for added depth.
*   **Lucide React**: Clean, sharp iconography for functional and decorative elements.

## 🚀 LOCAL DEVELOPMENT

To launch the architectural engine in your local atelier:

1.  **Clone the Vision**
    ```bash
    git clone https://github.com/agnibytes/AT-Mordern.git
    cd AT-Mordern
    ```

2.  **Initialize Dependencies**
    *(Requires `--legacy-peer-deps` to handle the bleeding-edge React 19 / Next.js 16 setup)*
    ```bash
    npm install --legacy-peer-deps
    ```

3.  **Flash the Build**
    ```bash
    npm run dev
    ```
    Live at: `http://localhost:3000`

## 💡 ENGINEERING INSIGHTS

*   **Hydration Mastery**: Every dynamic interaction (particles, stickers, random offsets) is protected by **Mounted Guards** to ensure perfect hydration between server and client.
*   **Animation Discipline**: All transitions use strictly audited **Framer Motion Easing** (easeOut) or native **GSAP Power Strings** to prevent runtime conflicts.
*   **Performance Awareness**: Optimized with `requestAnimationFrame` and component pooling to maintain 60FPS fluid motion across complex shader layers.
